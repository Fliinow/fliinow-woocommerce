WooCommerce payment gateway that integrates Fliinow as an installment financing option at checkout.

## Installation

### 1. Copy the plugin

```bash
cp -r fliinow-checkout /path/to/wordpress/wp-content/plugins/
```

### 2. Build JavaScript (required for WooCommerce Blocks)

```bash
cd fliinow-checkout
npm install
npm run build
```

### 3. Activate in WordPress

Go to **Plugins** → **Activate** "Fliinow – Checkout Financing"

### 4. Configure

Go to **WooCommerce → Settings → Payments → Fliinow** and set up:

| Field | Description |
|---|---|
| **API Key** | Your `fk_test_*` (sandbox) or `fk_live_*` (production) key |
| **Sandbox Mode** | Enable for testing with `demo.fliinow.com` |
| **Minimum Amount** | Minimum cart total to show financing (default: €60) |
| **Maximum Amount** | Maximum cart total (0 = no limit) |
| **Package Travel** | Mark as package travel (EU Directive) |

## Payment Flow

```
Customer checkout → Selects "Finance with Fliinow"
       ↓
Plugin creates operation via POST /operations
       ↓
Redirects to financingUrl (Fliinow)
       ↓
Customer picks a plan and completes the application
       ↓
Fliinow redirects to successCallbackUrl or errorCallbackUrl
       ↓
Plugin verifies real status via GET /operations/{id}/status:
  - FAVORABLE/CONFIRMED/FINISHED → order completed ✓
  - PENDING/GENERATED            → order on-hold (cron retries)
  - REFUSED/EXPIRED/ERROR        → order cancelled
  - API unreachable / no data    → order on-hold (cron retries)
```

## Extensibility

### Filter: `fliinow_wc_operation_data`

Modify the data sent to the Fliinow API before creating an operation:

```php
add_filter( 'fliinow_wc_operation_data', function( $data, $order ) {
    // Add flight data
    $data['flightDtoList'] = [
        [
            'origin'       => 'MAD',
            'destination'  => 'CDG',
            'flightType'   => 'ROUND_TRIP',
            'startDate'    => '15-06-2026',
            'endDate'      => '22-06-2026',
            'price'        => 450.00,
            'passengerList' => [
                [
                    'firstName' => $order->get_billing_first_name(),
                    'lastName'  => $order->get_billing_last_name(),
                    'birthDate' => '15-03-1985',
                ],
            ],
            'segments'     => [
                [
                    'flightNumber'       => 'IB1234',
                    'airlineCode'        => 'IB',
                    'originAirport'      => 'MAD',
                    'destinationAirport' => 'CDG',
                    'departureDateTime'  => '2026-06-15T08:00:00',
                    'arrivalDateTime'    => '2026-06-15T10:30:00',
                ],
            ],
        ],
    ];

    // Change number of travelers
    $data['travelersNumber'] = 2;

    return $data;
}, 10, 2 );
```

### Custom Client Fields

The plugin reads these order meta fields to populate required Fliinow client data:

| Meta key | Description | Example |
|---|---|---|
| `_billing_document_id` | ID / Passport number | `12345678A` |
| `_billing_document_validity` | Document expiry date | `31-12-2030` |
| `_billing_gender` | Gender (`MALE` / `FEMALE`) | `MALE` |
| `_billing_birth_date` | Date of birth | `15-03-1985` |

You can add these fields to checkout using plugins like "Checkout Field Editor" or programmatically:

```php
add_filter( 'woocommerce_checkout_fields', function( $fields ) {
    $fields['billing']['billing_document_id'] = [
        'type'     => 'text',
        'label'    => 'ID / Passport',
        'required' => true,
        'priority' => 35,
    ];
    $fields['billing']['billing_birth_date'] = [
        'type'     => 'date',
        'label'    => 'Date of birth',
        'required' => true,
        'priority' => 36,
    ];
    return $fields;
} );

// Save custom fields to order meta
add_action( 'woocommerce_checkout_update_order_meta', function( $order_id ) {
    $order = wc_get_order( $order_id );
    if ( ! empty( $_POST['billing_document_id'] ) ) {
        $order->update_meta_data( '_billing_document_id', sanitize_text_field( $_POST['billing_document_id'] ) );
    }
    if ( ! empty( $_POST['billing_birth_date'] ) ) {
        $order->update_meta_data( '_billing_birth_date', sanitize_text_field( $_POST['billing_birth_date'] ) );
    }
    $order->save();
} );
```

```bash
composer install
```

```bash
# PHP unit tests (128 tests, 257 assertions)
vendor/bin/phpunit --testsuite=unit

# Integration tests against sandbox (requires API key)
vendor/bin/phpunit --testsuite=integration

# JS tests (17 tests)
npm run test:js

npm run build
```

Copy `phpunit.xml.dist` to `phpunit.xml` and set your sandbox API key in `FLIINOW_TEST_API_KEY`.

## Distribution

### Option 1: Direct download (ZIP)

Install the plugin from GitHub Releases:

1. Go to [Releases](../../releases) and download `fliinow-checkout-X.X.X.zip`
2. In WordPress: **Plugins → Add New → Upload Plugin** → select the ZIP
3. Activate and configure at **WooCommerce → Settings → Payments → Fliinow**

### Option 2: Composer (for developers)

```json
{
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/Fliinow/fliinow-checkout"
    }
  ],
  "require": {
    "fliinow/fliinow-checkout": "^1.0"
  }
}
```

